## How To Run

From your command line, start with cloning the repo:

```bash
# Clone this repository
$ git clone https://github.com/hkurma/my-portfolio

# Go into the repository
$ cd my-portfolio
```

Now, install the NPM dependencies

```bash
# Installing NPM dependencies
$ npm i
```

Runing development version of the site

```bash
# Run the site
$ gatsby develop
```

At this point, the site should be up and running. To access it navigate to `http://localhost:8000/`
